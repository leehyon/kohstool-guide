## TL;DR
A terminal file manager that works like ranger but with vim like keybindings

## 应用场景
- 在终端中快速浏览文件系统
- 使用类似 vim 的快捷键管理文件
- 在不离开终端的情况下进行文件操作
- 需要轻量级文件管理器的开发者场景
- 喜欢键盘操作的用户日常文件管理

## 用户痛点
- 传统终端文件操作需要输入复杂命令
- 在图形界面和终端之间切换效率低下
- 寻找轻量级但功能完善的终端文件管理器
- 需要快速预览文件内容而不离开终端
- 希望减少鼠标使用，提高操作效率

## 设计理念
- 键盘优先
- 类似 vim 的操作逻辑
- 轻量级设计
- 保持终端环境下的流畅体验
