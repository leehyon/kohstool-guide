## TL;DR
一款基于 Web 的 Cloudflare R2 桶文件管理器，无需安装，在浏览器中直接操作文件

## 应用场景
- 管理 Cloudflare R2 存储桶中的文件
- 作为私有图床使用，拖拽上传图片
- 在浏览器中预览和编辑存储的文件
- 通过 PWA 将应用安装到桌面使用
- 在不同设备间同步文件管理配置

## 用户痛点
- 官方控制台功能基础，操作流程复杂
- 第三方客户端需要下载安装，跨平台使用麻烦
- 命令行工具学习门槛高，不适合临时操作
- 其他 Web 项目不专注 R2，功能不完善
- 需要频繁上传下载文件，缺乏便捷工具

## 设计理念
- 零构建，源码即产物
- 零框架，原生 Web 技术优先
- 零后端，所有逻辑在浏览器中完成
- 性能至上，懒加载、防抖节流
