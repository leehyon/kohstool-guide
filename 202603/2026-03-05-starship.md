## TL;DR
一个跨 shell 的提示符工具，提供快速、可高度定制化的终端体验

## 应用场景
- 为不同 shell 提供统一的终端提示符体验
- 自定义终端提示符显示信息和样式
- 提高终端工作效率和视觉体验
- 在多种操作系统间保持一致的 shell 体验
- 通过 Rust 实现高性能的命令行提示

## 用户痛点
- 不同 shell 提示符格式不统一，需要分别配置
- 默认 shell 提示符功能有限，无法显示丰富信息
- 自定义 shell 提示符需要编写复杂脚本
- 终端提示符性能不佳影响工作效率
- 跨平台使用时提示符显示不一致

## 设计理念
- 跨平台兼容性
- 高性能与可靠性
- 高度可定制化
- 简洁与功能平衡
